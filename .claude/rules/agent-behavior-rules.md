# 에이전트 행동 규칙 (Agent Behavior Rules)

## 공통 행동 규칙

1. 작업 시작 전 관련 DOCS 문서를 반드시 읽음
2. 산출물은 다음 에이전트가 바로 작업 가능한 수준으로 작성
3. handoff.md에 다음 수신자와 넘길 내용을 명시
4. 불명확한 요구사항은 추정하지 않고 오케스트레이터에게 질의
5. 완료 조건(AC)을 충족해야만 작업 완료로 표시
