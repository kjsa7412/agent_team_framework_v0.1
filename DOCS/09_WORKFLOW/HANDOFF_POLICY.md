# Handoff 정책 (Handoff Policy)

## Handoff 정의

에이전트 간 작업 인계. 산출물과 다음 에이전트 지시 포함.

## Handoff 필수 항목

모든 handoff.md에 포함:
- 보내는 에이전트
- 받는 에이전트
- 완료된 산출물 목록
- 다음 에이전트가 받아야 할 입력
- 주의사항/제약사항
- 완료 조건(AC) 확인 여부

## Handoff 검증

handoff-check 스킬로 검증:
- 필수 산출물 존재 여부
- AC 충족 여부
- 다음 에이전트 입력 준비 여부

## 오케스트레이터 역할

모든 handoff는 오케스트레이터가 통제.
오케스트레이터 없이 에이전트 간 직접 handoff 금지.
