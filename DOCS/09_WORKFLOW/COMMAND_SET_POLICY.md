# 명령 세트 정책 (Command Set Policy)

## 명령 체계

관리자가 .claude/commands/admin/ 내 명령으로 프레임워크를 지시.

## 명령 목록

| 명령 | 설명 |
|------|------|
| /work-new | 새 Work Item 생성 |
| /work-start | Work Item 실행 시작 |
| /work-status | Work Item 상태 조회 |
| /work-review | Work Item 검토 요청 |
| /work-approve | Work Item 승인 |
| /work-close | Work Item 종료 |
| /foundation-run | Foundation Group 실행 |
| /evolution-run | Evolution Group 실행 |
| /intelligence-run | Intelligence Group 실행 |
| /handoff | 에이전트 간 handoff 처리 |
| /promote-agent | 에이전트 승격 |
| /promote-skill | 스킬 승격 |
| /deprecate-agent | 에이전트 퇴역 |
| /deprecate-skill | 스킬 퇴역 |
| /policy-change-request | 정책 변경 요청 |

## 명령 기록

모든 명령 실행은 docs/00_admin/command-history.md에 기록.
