# 용어 사전 (Glossary)

| 용어 | 정의 |
|------|------|
| Work Item | 하나의 요청/문제를 해결하기 위한 실행 단위 |
| Orchestrator | 그룹 내 전체 흐름을 조정하는 에이전트 |
| Foundation Group | 최초 기획안을 받아 서비스 뼈대를 만드는 그룹 |
| Evolution Group | 운영 중 서비스에 요구사항을 반영하는 그룹 |
| Intelligence Group | 운영 데이터를 분석하는 그룹 |
| Test Group | 독립 테스트 조직 |
| Handoff | 에이전트 간 작업 인계 |
| AC | Acceptance Criteria (완료 조건) |
| RFP | Request for Proposal (기획 요청서) |
| evolution-small | 작은 버그/기능 보강, 구조 변화 없음 |
| evolution-large | 범위 크지만 기존 구조 위에서 처리 가능 |
| foundation-rebuild | 기존 구조 재정비 필요 |
| hybrid | 구조 재설계 + 기능 반영 동시 필요 |
| intelligence-only | 분석 중심 작업 |
| active | 공식 운영 중 상태 |
| candidate | 유효성 검증 중 상태 |
| deprecated | 중심 구조에서 밀려난 상태 |
| archive | 장기간 미사용 또는 대체재 존재 |
| DTO | Data Transfer Object (본 프레임워크에서 미사용, Map 기반 사용) |
| Monorepo | Front/Back/Tests/Scripts/Docs를 한 저장소에서 관리 |
