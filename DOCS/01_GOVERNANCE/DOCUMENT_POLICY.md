# 문서 정책 (Document Policy)

## 문서 원칙

1. 모든 핵심 산출물은 다음 에이전트의 작업지시서 역할을 해야 함
2. 산출물에는 반드시 "다음에 누가 무엇을 해야 하는지"가 명시되어야 함

## 산출물 필수 포함 항목

- 목적
- 이번 단계에서 반드시 해야 할 일
- 하지 말아야 할 일
- 입력 문서
- 선행 조건
- 세부 작업 항목
- 완료 조건(AC)
- 반환해야 할 산출물
- 다음 수신자에게 넘길 메모

## 문서 저장 위치

| 문서 유형 | 저장 위치 |
|----------|----------|
| 전역 정책 | DOCS/ |
| Work Item 실행 기록 | docs/work-items/{WI-ID}/ |
| 에이전트 정의 | .claude/agents/ |
| 명령 정의 | .claude/commands/ |
| 스킬 정의 | .claude/skills/ |
