# 승인 정책 (Approval Policy)

## 승인 필요 항목

| 항목 | 승인 주체 |
|------|----------|
| foundation-rebuild 실행 | 관리자 |
| 공통 아키텍처 변경 | 관리자 |
| 보안 정책 변경 | 관리자 |
| 정책 승격 (candidate → active) | 관리자 |
| 에이전트 승격/퇴역 | 관리자 |
| 프로덕션 배포 | 관리자 |

## 승인 기록

모든 승인은 docs/00_admin/approvals.md 에 기록한다.

## evolution-small 예외

evolution-small 분류 작업은 관리자 사전 승인 없이 진행 가능.
단, 완료 후 결과 보고는 필수.
